# ✨ **Isekai Character Card Generator** ✨

Welcome to the **Isekai Character Card Generator**! 🎮🌟  
Have you ever wondered what your fate would be if you were summoned to another world? Well, now you can find out! This app lets you create **your very own Isekai character card** as if you were summoned to a fantasy realm.

### ⚔️ **Features**
- **Summon Your Fate:** Choose your role, cheat skill, and quest to get your card.
- **Dynamic “Plot Armor”**: Don’t worry, you won’t die in the first 5 minutes (unless you’re unlucky). 
- **Guild Registration**: Get a magical Isekai ID, complete with your unique stats.
- **Customizable Fields**: Fill out the details, or let the card **taunt you** if you leave them blank. (Hey, it's all part of the adventure!)  
- **Unique Card Designs**: With an anime-style UI and a sprinkle of magic, your card will stand out. 

### 💀 **Taunt-Worthy Default Values**
- **Role:** “Cannon Fodder (RIP)” if you don’t fill it out.  
- **Cheat Skill:** “Can trip over destiny itself.”  
- **Quest:** “Surviving the first 5 minutes… maybe.”  
- **Plot Armor:** “0 stars – don’t count on surviving.”

### 🔮 **How to Use**
1. **Fill out your details** (role, cheat skill, etc.).
2. **Hit "Summon Your Fate"** to generate your magical Isekai character card.  
3. If you leave any fields blank, don’t worry! The card will taunt you with some **fun, anime-style default values**.
4. **Download and share your card** or use it as your new guild membership card!

### 🛠️ **Running the Project Locally**
1. Clone this repo:
    ```bash
    git clone https://github.com/your-username/isekai-character-card.git
    cd isekai-character-card
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Run the project:
    ```bash
    npm start
    ```

4. Open your browser and go to **http://localhost:3000** to generate your character card!
